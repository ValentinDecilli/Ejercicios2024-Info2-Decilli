    El programa consiste en resolver problemas basicos de caida libre, movimiento en dos dimensiones y movimiento circular uniforme.
Se le solicitara al usuario a traves de un menu, ingresar la opcion que corresponda segun el tipo de movimiento en el que se 
encuentre el objeto. El usuario debe ingresar los datos que conozcas, en caso de no conocerlo debera ingresar el caracter que 
corresponde. A traves de distintas funciones se corregiran al usuario en caso de ingresar valores que no correspondan. Para las 
funciones se utilzaron las formulas vistas en 1er a√±o. Hay que tener en cuenta el sistema de referencia que se este utilizando. 
Para nombrarar a las variables se utilizan estructuras y se las ennumera segun el movimiento que realize.

El programa tiene una estructura basica a seguir:

1- Seleccion del movimiento (void eleccion).
2- Escaneo de las variables.
3- Serie de calculos para la obtencion de las variables faltantes.  
4- Lectura de datos obtenidos.
